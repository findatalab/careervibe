# CareerVibe — карьерный нетворкинг

Веб-платформа для студентов и молодых специалистов, где можно задать вопрос профессионалам и получить карьерную консультацию. Проект разработан в рамках лаборатории искусственного интеллекта [Финансового университета при Правительстве РФ](www.fa.ru).

## Технологии

- **Backend:** Python 3, Flask
- **Database:** SQLite3
- **Frontend:** HTML5, CSS3 (чистый, без фреймворков)
- **Auth:** Werkzeug (хеширование паролей, сессии)

## Функционал

- Регистрация и авторизация пользователей (email + пароль)
- Хеширование паролей
- Сессии с проверкой валидности
- Лендинг с лентой вопросов
- Адаптивная вёрстка
- Выезжающие уведомления (toast) при входе/выходе/ошибках

## Установка и запуск

### Требования

- Python 3.10 или выше
- pip

### Шаги

1. **Клонировать или распаковать архив**

```bash
cd careervibe
```

2. **Создать и активировать виртуальное окружение**

```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux / macOS:
source venv/bin/activate
```

3. **Установить зависимости**

```bash
pip install -r requirements.txt
```

4. **Инициализировать базу данных**

```bash
python init_db.py
```

5. **Запустить приложение**

```bash
python app.py
```

6. **Открыть в браузере**

```
http://127.0.0.1:5000
```

## Структура проекта

```
careervibe/
├── app.py                 # Основное приложение Flask
├── init_db.py             # Скрипт создания БД
├── requirements.txt       # Зависимости
├── instance/
│   └── careervibe.db      # База данных (создаётся при init_db)
├── static/
│   └── style.css          # Все стили
└── templates/
    ├── base.html          # Базовый шаблон (шапка, футер, тосты)
    ├── index.html         # Главная страница (лендинг + лента)
    ├── login.html         # Вход и регистрация
    └── mission.html       # Страница «Миссия»
```

## Примечания

- Секретный ключ Flask можно изменить в `app.py` или через переменную окружения `SECRET_KEY`
- База данных создаётся в папке `instance/`
- После регистрации в шапке появляются аватар (первая буква email) и кнопка выхода
